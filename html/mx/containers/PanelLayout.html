mx.containers

PanelLayout 类

全称 mx.containers.PanelLayout
命名空间 mx.containers
类 PanelLayout
继承层次结构 PanelLayoutContainerControl
源文件 ~\mx\scripts\containers\PanelLayout.js
提供一个多面板布局的控件。面板有上下左右中五个，可使用其中任意组合的面板。

示例 1

以下是一个创建 PanelLayout 的示例。
JavaScript 代码
var layout = new mx.containers.PanelLayout({ topPanelVisible:false, // 隐藏顶部面板 leftPanelVisible:false, // 隐藏左部面板 middlePanelHeight:400, // 设置中部面板高度 middlePanelWidth:"50%" // 设置中部面板宽度 });

示例 2

以下是一个向 PanelLayout 控件的面板中添加内容的示例。
JavaScript 代码
var btn = new mx.controls.Button({ text:"bruce" }); var label = new mx.controls.Label({ text:"dbm" }); layout.addControl(label); // 添加到中部面板 layout.addControl(btn, "top"); // 添加到顶部面板

参数

  • p_position
    一个字符串， 表示需要隐藏或显示的面板位置。

    该参数常见值包括:

    • left （隐藏或显示左边面板）
    • top （隐藏或显示顶部面板）
    • right （隐藏或显示右边面板，前提是中间区域分为了左、中、右三部分）
    • bottom（隐藏或显示底部面板）

mx.controls.Control.visible

默认值

true

width 字段

获取容器的宽度。可以是数字（如 500）、字符串（如“500px”或“50%”）。

继承自

mx.containers.Container.width

默认值

auto

$container 字段

获取容器控件中用来包含控件 DOM 元素的占位符。

继承自

mx.containers.Container.$container

$e 字段

获取该控件的占位符，该占位符为一个 JQuery 对象。

继承自

mx.controls.Control.$e
