mx.editors 命名空间

全称 mx.editors
提供一组编辑器控件。 所有编辑器的基类均为 Editor。 编辑器与表单（DataForm）、数据源（EntityContainer）等进行配合，可以实现数据绑定，从而实现数据展现、编辑的自动化。

类

名称 说明
AutoCompleteEditor 智能查询控件。
CheckEditor 提供一个显示允许用户选择 true 或 false 条件的复选框的类。
CheckListEditor 一个提供组合单选或者复选项的编辑器类。
CheckListItem 提供一个多选项或单选项列表控件（CheckListEditor 类型）中的数据项的类。请不要单独使用。
ComboEditor 提供一个带有下拉列表的选择控件，通过单击控件上的箭头可显示或隐藏下拉列表。该类的派生类包含：DateTimeEditor、DropDownEditor 等。
ComplexConditionEditor 提供一个设置复杂条件的编辑器。
ConditionFilterEditor 提供一个下拉的高级查询控件。该控件与 DataGridSearchBox 配合使用。
CustomEditor 提供一个用户自定义按钮方法的文本编辑器。
DateTimeEditor 提供一个下拉的日期选择控件。
DropDownEditor 提供一个允许用户从下拉列表中选择数据项的控件的类。该控件支持可编辑和不可编辑两种模式，当 DropDownEditor.displayMember 和 DropDownEditor.valueMember 相同并且 ComboEditor.allowEditing 为 true 时，控件支持编辑；当 DropDownEditor.displayMember 和 DropDownEditor.valueMember 不同时，控件不支持编辑（ComboEditor.allowEditing 属性失效）。
DropDownGridEditor 提供一个下拉表格的类。
DropDownTreeEditor 提供一个允许用户从下拉树中选择数据项的控件的类。支持单选和多选。
Editor 提供所有编辑器的抽象基类。该类的派生类包含：CheckEditor、ComboEditor、ListEditor、UpDownEditor、TextEditor 等。
EditorFactoryClass 提供编辑器工厂类。该类在运行时唯一静态实例是 mx.editors.EditorFactory 对象。详见 EditorFactoryClass.createEditor 方法示例。
FileEditor 提供一个显示上传文件信息的类。
LabelEditor 提供一个显示文字的标签编辑器（通常该编辑器只是用于显示，不提供编辑功能）。
LinkEditor 提供一个允许用户点击进行页面跳转或其他操作的链接编辑器的类。
ListEditor 提供一个允许用户进行数据选择操作的列表编辑器的类。
ListItem 提供一个在 ListEditor 控件中显示的列表项的类。另请参见 ListEditor类。
NumberEditor 提供一个数字编辑器类。
PictureEditor 提供一个允许用户点击选择图片的类。
PictureManagerEditor 提供一个上传图片的类。
RichTextEditor 提供一个允许用户输入带格式文本的富文本编辑器的类。
SliderEditor 提供一个允许用户拖动进度的类。
TextEditor 提供一个允许用户输入文本的文本框的类。
TimeEditor 提供一个时间编辑控件。
UpDownEditor 提供一个数字编辑器和时间编辑器的基类。该类的派生类包含：NumberEditor、TimeEditor。
