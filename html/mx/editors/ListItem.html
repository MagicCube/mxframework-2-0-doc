ListItem 类

全称 mx.editors.ListItem
命名空间 mx.editors
类 ListItem
继承层次结构 ListItem
源文件 ~\mx\scripts\editors\ListEditor.js
提供一个在 ListEditor 控件中显示的列表项的类。另请参见 ListEditor类。

构造函数

名称 说明
ListItem() 构建 ListItem 类的新实例。

初始化函数

名称 说明
init() 初始化数据项。

方法

名称 说明
getIndex() 返回该对象在 ListEditor 的 ListEditor.items 集合中的序号（0 代表第一个）。如果该对象的 owner 字段为空，则返回 -1。
remove() 删除当前项。该方法仅当 owner 字段不为空时有效。
scrollIntoView() 设置 ListEditor 的滚动条的顶部偏移值（该对象容器到父容器的顶部偏移值）。
setChecked(p_checked) 设置 checked 字段的值。
setText(p_text) 设置 text 字段的值。
setValue(p_value) 设置 value 字段的值。
toggleChecked() 切换选中状态。

字段

名称 说明
checked 获取一个值，表示该对象是否处于选中状态。如果该值为 true，则表示选中；反之则不选中。 请使用 setChecked 方法设置该字段的值。
owner 获取该对象的父容器的对象，通常是一个 ListEditor 类的实例。
text 获取该对象的显示值。请使用 setText 方法设置该字段的值。
value 获取该对象的实际值。该字段也是该对象在 ListEditor 所有项中的唯一标识。请使用 setValue 方法设置该字段的值。注意该字段不能是数字。
