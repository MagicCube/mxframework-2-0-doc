mx.weblets 命名空间

全称 mx.weblets
提供 Web 小程序相关的容器、上下文、加载器、管理类的控件。

类

名称 说明
Weblet 描述一个 Web 小程序的类。请不要在程序中直接实例化该类。 通常该类会在 weblet.js 中通过 WebletManagerClass.register 方法创建。
WebletContainer 提供一个 Weblet 的容器。继承自 ViewPort。
WebletContext 描述 Weblet 的上下文信息，包括唯一标识（id）、名称以及自定义描述信息。 通常 Weblet 本身可以通过 Weblet.context 属性获得。如果是继承自 Control 的类的实例可以通过 Control.getContext 方法获得。
WebletLoaderClass 提供一个 Weblet 加载器的类。该类只在 MX 框架内部使用，不建议开发人员直接使用。
WebletManagerClass 提供 Weblet 管理的类。该类在运行时唯一的静态实例是 mx.weblets.WebletManager 对象。
