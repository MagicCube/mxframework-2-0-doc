DataRow 类

全称 mx.datas.DataRow
命名空间 mx.datas
类 DataRow
继承层次结构 DataRowDataObject
源文件 ~\mx\scripts\datas\DataTable.js
表示数据表 DataTable 中数据行的类。

示例 1

以下为 DataRow 的示例。说明了如何创建一个 DataRow。
JavaScript 代码
var r = new mx.datas.DataRow(); var r2 =new mx.datas.DataRow("张三", 23, "男", "1984"); var r3 =new mx.datas.DataRow(["张三", 23, "男", "1984"]); 　　var r4 = new mx.datas.DataRow({v:"张三"},{v:23},{v:1,f:"男"},{v:1987}); var r5 = new mx.datas.DataRow({value:"张三"},{value:23},{value:1,formattedValue:"男"},{value:"1987"}); var r6 = new mx.datas.DataRow({cells:["张三", 23, "男", "1984"]}); 　　var r7 = new mx.datas.DataRow({cells:[{v:"张三"},{v:23},{v:1,f:"男"},{v:1987}]}); var r8 = new mx.datas.DataRow({cells:[{value:"张三"},{value:23},{value:1,formattedValue:"男"},{value:"1987"}]});
