DataColumn 类

全称 mx.datas.DataColumn

类 DataColumn

表示数据表 DataTable 中数据列 DataColumn 的类。

示例 1

以下为 DataColumn 的示例。说明了如何创建一个 DataColumn。
JavaScript 代码
var c =new mx.datas.DataColumn(); var c2 =new mx.datas.DataColumn("name", "名字", "string"); var c3 =new mx.datas.DataColumn("name", "名字", "string","yyyy-MM-dd"); var c4 =new mx.datas.DataColumn({id:"name", label:"名字", type:"string"}); var c5 =new mx.datas.DataColumn(["name", "名字", "string","yyyy-MM-dd"]);
