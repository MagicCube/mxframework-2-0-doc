ToolSeparator 类

全称 mx.controls.ToolSeparator
命名空间 mx.controls
类 ToolSeparator
继承层次结构 ToolSeparatorObject
源文件 ~\mx\scripts\controls\ToolSeparator.js
表示分割线，用于对 ToolStrip 的项或者 MainMenu、 ContextMenu 控件的子项进行分组。

构造函数

名称 说明
ToolSeparator(p_name) 构建 ToolSeparator 类的新实例。

初始化函数

名称 说明
init() 初始化。

方法

名称 说明
hide() 将该 ToolSeparator 置为不可见，即将 visible 字段置为 false。
setVisible(p_visible) 设置 visible 字段的值。
show() 将该 ToolSeparator 置为可见，即将 visible 字段置为 true。

字段

名称 说明
isGroup 获取一个 Boolean 值，表示该项是否包含子项。如果该值为 true，表示有子项；反之则没有。
isSeparator 获取该项是否为分隔符。如果为 true，表示该项为分隔符；反之则不是。
name 获取或设置项的名称。
owner 获取该项的父容器的对象，通常是一个 ToolStrip 类型的实例。
parent 获取该项的父 ToolStripItem 对象。如果没有，则为 null。
visible 获取该项是否为可见状态。如果该值为 true，则表示可见；反之则不可见。
