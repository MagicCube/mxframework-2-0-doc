Slideshow 类

全称 mx.controls.Slideshow
命名空间 mx.controls
类 Slideshow
继承层次结构 Slideshow
源文件 ~\mx\scripts\controls\Slideshow.js
提供一个幻灯片预览控件的类。

示例 1

以下是一个创建 Slideshow 的示例。 该示例将说明如何创建一个 Slideshow。
JavaScript 代码
var slider = new mx.controls.Slideshow({ auto:true, direction: "toLeft", items:[{src:"http://gsky:8080/mxframe/dbm/resources/images/workbench.png"}, {src:"http://gsky:8080/mxframe/dbm/resources/images/login.png"}, {src:"http://gsky:8080/mxframe/dbm/resources/images/mx.png"}] }); var button = new mx.controls.Button({ text: "show", onclick: function(){ slider.show(); } });

构造函数

名称 说明
Slideshow() 构建 Slideshow 类的新实例。

方法

名称 说明
setAuto([p_auto]) 设置 auto 字段的值。
setPause(p_pause) 设置 pause 字段的值。
show(p_startIndex) 提供一个方法，开始预览。本方法是幻灯片预览的入口。
showNext(p_startIndex) 提供一个方法，预览下一张幻灯片。
showPre() 提供一个方法，预览上一张幻灯片。
slide([p_slide]) 提供一个方法，滑动预览，或者停止预览，当 auto 字段的值为 true 时可以滑动预览。

字段

名称 说明
auto 获取一个 Boolean 值，表示幻灯片是否自动滑动。请使用 setAuto 方法设置该字段的值。
currentItem 获取一个 SlideshowItem 对象， 表示当前滑动的幻灯片，如果一个滑入，一个滑出，则表示滑入的。
direction 设置一个字符串，表示幻灯片滑动的方向。

该字段可选的值包括：

  • toRight
  • toLeft
items 获取和设置一个集合，用于创建含控件中要展示的 SlideshowItem。
pause 获取一个数字，表示幻灯片滑动的停留时间，单位为毫秒。请使用 setPause 方法设置该字段的值。
pauseTimes 获取一个数组，表示是否显示播放按钮可选择的幻灯片停留时间，请在初始化之前设置该字段的值。
playButton 获取一个 Boolean 值，表示是否显示播放按钮，请在初始化之前设置该字段的值。
speed 获取或设置一个数字，表示幻灯片滑动的速度，单位为毫秒。
wizardButton 获取一个 Boolean 值，表示是否显示左右按钮，请在初始化之前设置该字段的值。

方法详细信息

setAuto([p_auto]) 方法

设置 auto 字段的值。

参数

  • p_auto
    可选的。默认值为“true”。一个 Boolean 值，表示幻灯片是否自动滑动。

setPause(p_pause) 方法

设置 pause 字段的值。

参数

  • p_pause
    一个数字，表示幻灯片停留的时间，单位为毫秒。

show(p_startIndex) 方法

提供一个方法，开始预览。本方法是幻灯片预览的入口。

参数

  • p_startIndex

showNext(p_startIndex) 方法

提供一个方法，预览下一张幻灯片。

参数

  • p_startIndex

showPre() 方法

提供一个方法，预览上一张幻灯片。

slide([p_slide]) 方法

提供一个方法，滑动预览，或者停止预览，当 auto 字段的值为 true 时可以滑动预览。

参数

  • p_slide
    可选的。默认值为“true”。一个 Boolean 值，值为 true 时，开始滑动预览；为 false 时，停止滑动。

字段详细信息

auto 字段

获取一个 Boolean 值，表示幻灯片是否自动滑动。请使用 setAuto 方法设置该字段的值。

默认值

true

currentItem 字段

获取一个 SlideshowItem 对象， 表示当前滑动的幻灯片，如果一个滑入，一个滑出，则表示滑入的。

direction 字段

设置一个字符串，表示幻灯片滑动的方向。

该字段可选的值包括：

  • toRight
  • toLeft

items 字段

获取和设置一个集合，用于创建含控件中要展示的 SlideshowItem。

pause 字段

获取一个数字，表示幻灯片滑动的停留时间，单位为毫秒。请使用 setPause 方法设置该字段的值。

默认值

2000

pauseTimes 字段

获取一个数组，表示是否显示播放按钮可选择的幻灯片停留时间，请在初始化之前设置该字段的值。

默认值

true

playButton 字段

获取一个 Boolean 值，表示是否显示播放按钮，请在初始化之前设置该字段的值。

默认值

true

speed 字段

获取或设置一个数字，表示幻灯片滑动的速度，单位为毫秒。

默认值

1000

wizardButton 字段

获取一个 Boolean 值，表示是否显示左右按钮，请在初始化之前设置该字段的值。

默认值

true
