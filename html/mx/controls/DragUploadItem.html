DragUploadItem 类

全称 mx.controls.DragUploadItem
命名空间 mx.controls
类 DragUploadItem
继承层次结构 DragUploadItem
源文件 ~\mx\scripts\controls\DragUploadControl.js
提供一个上传控件（DragUploadControl 类型）中的上传项的类。

构造函数

名称 说明
DragUploadItem() 构建 DragUploadItem 类的新实例。

初始化函数

名称 说明
init()

方法

名称 说明
cancel() 取消上传文件。
getName() 获取文件名。
getPath() 获取文件路径。
getSize() 获取文件大小。
getType() 获取文件类型。
setParams(p_params) 设置 params 的值。
setProgress(p_percentage) 设置 progress 字段的值。
setStatus(p_status) 设置 {status} 字段的值。
setUdsParam(p_params) 设置 params 的值。
upload(p_triggerEvent) 开始上传文件。

字段

名称 说明
file 获取或设置一个对象，表示要上传的文件。
fileUploader 获取一个 FileUploader 对象。
name 获取或设置一个字符串，表示文件名。
params 参见 FileUploader.params 字段。
path 获取或设置一个字符串，表示文件路径。
progress 获取或设置一个数字，表示上传的进度（如 50，表示上传了 50%）。
size 获取或设置一个数字，表示文件大小（单位：字节）。
status 获取或设置一个字符串，表示文件的上传状态 （DragUploadControl.uploadStatus.UNSTART，DragUploadControl.uploadStatus.UPLOADING，DragUploadControl.uploadStatus.COMPLETE， DragUploadControl.uploadStatus.STOP，DragUploadControl.uploadStatus.ERROR）。
type 获取或设置一个字符串，表示文件类型。

事件

名称 说明
onuploaded 文件上传之后触发该事件。
onuploading 文件上传之前触发该事件。
