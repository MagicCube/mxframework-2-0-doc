构造函数

方法

名称 说明

字段

名称 说明
isPlaying 获取一个 Boolean 值，表示 SlideshowItem 是否在滑动。

名称 说明
onplayed 幻灯片滑动完成时，将触发该事件。调用 paly 方法时会触发该事件。
onplaying 幻灯片开始滑动时，将触发该事件。调用 paly 方法时会触发该事件。

构造函数详细信息

SlideshowItem() 方法

构建 SlideshowItem 类的新实例。

方法详细信息

play(p_speed, p_fromStyle, p_animateStyle, p_complete) 方法

向 EntityContainer 滑动。

参数

  • p_speed
    一个数字，表示幻灯片滑动的速度，单位为毫秒。
  • p_fromStyle
    一个 JSON 对象，表示幻灯片滑动的开始位置。
  • p_animateStyle
    一个 JSON 对象，表示幻灯片滑动的终止位置。
  • p_complete

_init() 方法

受保护的。

字段详细信息

isPlaying 字段

获取一个 Boolean 值，表示 SlideshowItem 是否在滑动。

src 字段

设置或者获取一个字符串，表示幻灯片预览的图片路径，请在初始化之前设置该字段的值。

事件详细信息

onplayed 事件

幻灯片滑动完成时，将触发该事件。调用 paly 方法时会触发该事件。

参数

  • item
    一个 SlideshowItem 对象，表示当前滑动的幻灯片。

onplaying 事件

幻灯片开始滑动时，将触发该事件。调用 paly 方法时会触发该事件。

参数

  • cancel
    可通过设置 cancel 参数为 true 来取消当前操作。默认为 false。
  • item
    一个 SlideshowItem 对象，表示当前滑动的幻灯片。
  • fromStyle
    一个 JSON 对象，表示幻灯片滑动的开始位置。
  • animateStyle
    一个 JSON 对象，表示幻灯片滑动的终止位置。
