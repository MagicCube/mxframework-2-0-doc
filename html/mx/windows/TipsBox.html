mx.windows

TipsBox 类

全称 mx.windows.TipsBox
命名空间 mx.windows
类 TipsBox
继承层次结构 TipsBox
源文件 ~\mx\scripts\windows\TipsBox.js
提供一个Tips提示框的类。

示例 1

以下是一个创建 TipsBox 的示例。
JavaScript 代码
var TipsBox = new mx.windows.TipsBox({//参数可以new实例时时传入，也可以在调用show方法时传入。 title:"标题",//一个字符串，表示Tips的标题。 message:"内容",//一个字符串，表示Tips的内容。 showSpeed: 500,//一个数字值（毫秒），表示Tips显示的速度。 orientation : "bottom left",//两个字符串值，用空格格开，第一个值是向滚动方向，第二个值是tips框显示的绝对位置。 可用参数有："top right"|"bottom right"|"left top"|"right top"| "bottom left"|"top left"|"right bottom"|"left bottom"|"center center" width : 200,//一个数字型，表示第Tips宽。 height : 160,//一个数字型，表示第Tips高。 contentBackground:"#FFFFFF",//一个 css样式，表示Tips内容背景颜色样式。 autoHide:true,//一个 Boolean 对象，表示是否自动关闭Tips。 autoHideSpeed:2000 //一个数字值（毫秒），表示Tips关闭的速度。 }); TipsBox.show({ title : "Tips 标题", message : "这是Tips内容", orientation : "top right" });

字段

名称 说明

closeSpeed

message 获取或设置Tips提示框的内容。
orientation 获取或设置Tips提示框的滚动方向及位置。第一个参数是滚动方向，第二个是初始Tips位置。

该字段的值包括:

  • right bottom - 表示在左下角，向右移动。
