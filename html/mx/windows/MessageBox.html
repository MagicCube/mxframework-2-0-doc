在新窗口中打开隐藏基类成员
mx.windows

MessageBox 类

全称 mx.windows.MessageBox
命名空间 mx.windows
类 MessageBox
继承层次结构 MessageBox
源文件 ~\mx\scripts\windows\MessageBox.js
提供一个消息提示框的类。

示例 1

以下是一个使用消息提示框的示例。
JavaScript 代码
var msgBox = new mx.windows.MessageBox( { reusable : true,//一个Boolean值，表示弹出窗口是否可重用。 title : "标题",// 一个字符串，表示弹出窗口的标题。 showOkButton : true,//一个Boolean值，表示是否显示确定按纽。 showCancelButton : true,//一个Boolean值，表示是否显示取消按纽。 okButtonText : "确定",//一个字符串，表示第一个按纽名称。 cancelButtonText : "取消",// 一个字符串，表示第二个按纽名称。 message : "提示内容。",//一个字符串，表示弹出窗口的内容。 imgkey : "error",//一个字符串，表示弹出窗口的类型， 支持四种类型：info：提示信息 | warnning：警告信息 | error：错误信息 |prompt：输入框(如果类型是 prompt，可以调用时使用 function(){alert(this.textEditor.text);}来获取输入框的值) okButtonCallback : function() {//一个 Function 对象，表示第一个按钮的回调函数。 alert("第一个按纽回调") }, cancelButtonCallback : function() {//一个 Function 对象，表示第二个按钮的回调函数。 alert("第二个按纽回调") } }); msgBox.setMessage("setMessage"); msgBox.setTitle("setTitle"); msgBox.setOkButtonText("获取prompt的值"); msgBox.setCancelButtonText("canel"); msgBox.setImgkey("prompt"); msgBox.show( function() { if(this.textEditor!=null){ if(this.textEditor.text!=""){ alert(this.textEditor.text); }else{ alert("输入框值为空"); } }else{ alert("当前类型不是提示框"); } }, function() { alert("点击cancel的回调函数，此回调是在show方法里定义的"); });

方法

名称 说明
setCancelButtonText(p_text) 设置取消按钮的显示值。
setImgkey(p_imgkey) 设置 imgkey 字段的值。
setMessage(p_message) 设置 message 字段的值。
setOkButtonText(p_text) 设置确定按钮的显示值。
setTitle(p_title) 设置 title 字段的值。
show([p_okBtnCallback], [p_cancelBtnCallback]) 显示消息框。

字段

名称 说明
cancelButtonCallback 点击“取消”按钮后执行的回调函数。
cancelButtonText 获取或设置取消按钮的显示值。
height 获取或设置消息提示框的高度。
imgkey 获取或者设置提示框的类型，默认为：info。 支持四种类型：info：提示信息 | warnning：警告信息 | error：错误信息 | prompt：输入框。
message 获取或设置消息框显示内容。
okButtonCallback 点击“确定”按钮后执行的回调函数。
okButtonText 获取或设置确定按钮的显示值。
reusable 参见 Window.reusable 字段。
showCancelButton 获取或设置取消按钮是否显示，默认为显示。
showOkButton 获取或设置确定按钮是否显示，默认为显示。
title 获取或设置消息框标题。
width 获取或设置消息提示框的宽度。
zIndex 获取或设置窗体的zIndex值，控制窗口的层叠顺序，值大的窗口覆盖值小的窗口,值需大于0
