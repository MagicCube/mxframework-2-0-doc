FormField 类

全称 mx.datacontrols.FormField
命名空间 mx.datacontrols
类 FormField
继承层次结构 FormFieldObject
源文件 ~\mx\scripts\datacontrols\DataForm.js
提供一个表单字段信息的类。

构造函数

名称 说明
FormField() 构建 FormField 类的新实例。

字段

名称 说明
caption 获取字段的标题。 在运行时请使用 DataForm.setFieldCaption 方法改变该字段的值。
captionVisible 获取该字段的标题是否可见, 在运行时请使用 DataForm.setFieldCaptionVisible
colSpan 获取和设置字段列跨度属性。
editorType 获取字段的编辑器类型。
该字段的可选值包括:
  • CheckEditor
  • ComboEditor
  • DateTimeEditor
  • DropDownEditor
  • ListEditor
  • NumberEditor
  • PictureEditor
  • RichTextEditor
  • TextEditor
  • TimeEditor
  • UpDownEditor
  • SliderEditor
enabled 获取字段是否可用。 在运行时请使用 DataForm.setFieldEnabled 方法改变该字段的值。
fieldName 获取字段的名称。通常与 {
isVirtual 指示字段是否虚拟，如果字段是虚拟的话，其数据将不会与服务器交互
lineBreak 获取一个 Boolean 值，该值表示字段是否换行排列。
name 获取字段的名称。
nullable 获取字段是否可空。 在运行时请使用 DataForm.setFieldNullable 方法改变该字段的值。
readOnly 获取字段是否只读。 在运行时请使用 DataForm.setFieldReadOnly 方法改变该字段的值。
visible 获取字段是否可见。 在运行时请使用 DataForm.setFieldVisible 方法改变该字段的值。
width 获取字段的宽度。
