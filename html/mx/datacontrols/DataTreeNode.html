字段

名称 说明
childNodes 获取同步树中当前节点的子节点集合。
displayMode 获取一个字符串值，表示该节点的显示方式。

该字段的可选值包括:

  • entityNode - 实体节点。表示节点是一个实体对象节点。
  • propGroup - 属性分组节点。表示节点是属性分组。
  • virtualNode - 虚拟节点。表示节点是一个静态节点。
  • enumNode - 枚举节点。节点的值来源于枚举项
expanded 获取一个 Boolean 值，表示该节点的子节点是否展开。如果该值为 true，表示子节点展开；反之则隐藏。
hasChildren 获取一个 Boolean 值，表示该节点是否有子节点。如果该值为 true，则表示有子节点；反之则没有。
id 获取一个字符串，表示节点的唯一标识。请使用 setID 方法设置该字段的值。
imageKey 获取一个字符串，表示 DataTreeNode 图像的名称。请使用 setImageKey 方法设置该字段的值。
imageUrl 获取一个字符串，表示 DataTreeNode 图像的 URL（如“$/icons/role.png”）。请使用 setImageUrl 方法设置该字段的值。
isLasyLoad 获取或设置一个 Boolean 值，表示树的节点是否懒加载。
itemType 获取一个字符串，表示说明节点的类型（如“folder”，“role”，“job”等）。请使用 setItemType 方法设置该字段的值。
owner 获取该对象的父容器的对象，通常是一个 DataTree 类的实例。
parent 获取该对象的父节点对象，通常是一个 DataTreeNode 类的实例。
queryParams 一个 JSON 对象，通常在加载子节点时需要一些请求参数时，可以使用字段。 使用时可以订购 DataTree.onexpanding 事件，在树节点展开之前为该字段赋值。 该请求参数会与默认的请求参数合并后一起传到服务端。
text 获取节点显示的文本。请使用 setText 方法设置该字段的值。

_createNode(p_node) 方法

受保护的。根据指定的条件创建并返回节点（DataTreeNode 类型）。

参数

  • p_node
    一个 JSON 对象，表示 DataTreeNode 的一组属性值；或一个 DataTreeNode 对象。

字段详细信息

childNodes 字段

获取同步树中当前节点的子节点集合。

默认值

null

displayMode 字段

获取一个字符串值，表示该节点的显示方式。

该字段的可选值包括:

  • entityNode - 实体节点。表示节点是一个实体对象节点。
  • propGroup - 属性分组节点。表示节点是属性分组。
  • virtualNode - 虚拟节点。表示节点是一个静态节点。
  • enumNode - 枚举节点。节点的值来源于枚举项

默认值

null

expanded 字段

获取一个 Boolean 值，表示该节点的子节点是否展开。如果该值为 true，表示子节点展开；反之则隐藏。

默认值

false

hasChildren 字段

获取一个 Boolean 值，表示该节点是否有子节点。如果该值为 true，则表示有子节点；反之则没有。

默认值

false

id 字段

获取一个字符串，表示节点的唯一标识。请使用 setID 方法设置该字段的值。

imageKey 字段

获取一个字符串，表示 DataTreeNode 图像的名称。请使用 setImageKey 方法设置该字段的值。

imageUrl 字段

获取一个字符串，表示 DataTreeNode 图像的 URL（如“$/icons/role.png”）。请使用 setImageUrl 方法设置该字段的值。

isLasyLoad 字段

获取或设置一个 Boolean 值，表示树的节点是否懒加载。

默认值

true

itemType 字段

获取一个字符串，表示说明节点的类型（如“folder”，“role”，“job”等）。请使用 setItemType 方法设置该字段的值。

默认值

item

owner 字段

获取该对象的父容器的对象，通常是一个 DataTree 类的实例。

parent 字段

获取该对象的父节点对象，通常是一个 DataTreeNode 类的实例。

queryParams 字段

一个 JSON 对象，通常在加载子节点时需要一些请求参数时，可以使用字段。 使用时可以订购 DataTree.onexpanding 事件，在树节点展开之前为该字段赋值。 该请求参数会与默认的请求参数合并后一起传到服务端。

text 字段

获取节点显示的文本。请使用 setText 方法设置该字段的值。
