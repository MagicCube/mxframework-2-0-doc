mx.datacontrols 命名空间

全称 mx.datacontrols
提供数据控件，如 表格 DataGrid， 表单 DataForm，树 DataTree 等。 数据控件类用来加载列表，表单，和树的数据显示。

类

名称 说明
ComplexForm
ComplexGrid 提供一个集成工具栏和表格的复合组件。
DataForm 提供一个表单控件的类。
DataGrid 提供一个显示数据表格的控件。
DataGridColumn 提供一个数据项列表控件（DataGrid 类型）中列的类。 要与DataGrid配合使用，不能单独使用
DataGridItem 提供一个数据项列表控件（DataGrid 类型）中的数据项的类。该数据项容器的 HTML 元素为 <TR>。请不要单独使用。
DataGridItemGroup 提供一个行分组列表控件（GroupItemGrid 类型）中的分组的类。
DataGridSearchBox 提供一个查询框，它要和 PagedControl 配合使用。帮助有需要进行查询的的表格控件实现查询功能。
DataTree 提供一个在树状结构中显示分层数据的控件的类，该树状结构包含可展开和折叠的项。
DataTreeNode 提供一个 DataTree 控件中节点的类。
DynamicForm 动态表单，支持生成的容器包括 mx.containers.TabControl mx.containers.Accordion mx.containers.HSplit mx.containers.VSplit mx.containers.DockPanel 当验证表单输入时将触发该事件。调用 validate 方法时会触发该事件。可通过订购该事件，实现对表单的自定义验证。
DynamicFormField 提供一个表单字段信息的类。
FormField 提供一个表单字段信息的类。
FormFieldGroup 提供一个表单字段分组的信息的集合类。
GridColumnGroup 列分组
GroupHeaderGrid 提供一个显示多表头数据表格的控件。
GroupHeaderGridColumn 提供一个多表头数据项列表控件（GroupHeaderGrid 类型）中列的类。
GroupItemGrid 提供一个显示行分组表格的控件。
GroupItemGridColumn 提供一个行分组列表控件（GroupItemGrid 类型）中列的类。
GroupLockHeaderGrid 提供一个可以固定列，并支持多表头数据显示的数据表格控件。
LockHeaderGrid 提供一个可以固定列的数据表格的控件。
LockHeaderGridItem 提供一个数据项列表控件（LockHeaderGrid 类型）中的数据项的类。该数据项容器的 HTML 元素为 <TR>。
PagedControl 提供一个帮助需要进行分页的控件（如 DataGrid）进行分页的控件。
PageNaviBar 提供一个分页工具条，它要和 PagedControl 配合使用。帮助有需要进行分页的控件实现分页。
ScrollBar 滚动条控件
ScrollLoadGrid 提供一个支持数据滚动加载的DataGrid控件。
SkinDataForm 提供一个自定义模板的表单控件类。
