ClipboardUtilClass 类

全称 mx.utils.ClipboardUtilClass
命名空间 mx.utils
类 ClipboardUtilClass
继承层次结构 ClipboardUtilClassObject
源文件 ~\mx\scripts\utils\ClipboardUtil.js
提供剪贴簿的访问接口。该类在运行时唯一静态实例是 mx.utils.ClipboardUtil 对象。该接口仅对 Internet Explorer 浏览器有效。

示例 1

以下是一个通过 mx.utils.ClipboardUtil 获取和设置剪贴板中文本的示例。
JavaScript 代码
//获取剪贴板中的文本 var text = mx.utils.ClipboardUtil.getText(); //设置剪贴板中的文本 mx.utils.ClipboardUtil.setText("我是剪贴板中的文本");

构造函数

名称 说明
ClipboardUtilClass() 构建 ClipboardUtilClass 类的新实例。

方法

名称 说明
getText() 获取当前剪贴簿中存放的文本。如果浏览器不支持，则返回 null。
setText(p_text) 设置当前剪贴簿中的文本为 p_text。

方法详细信息

getText() 方法

获取当前剪贴簿中存放的文本。如果浏览器不支持，则返回 null。

setText(p_text) 方法

设置当前剪贴簿中的文本为 p_text。

参数

  • p_text
    一个文本，表示需要放入剪贴簿的文本。
