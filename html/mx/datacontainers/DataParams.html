DataParams 类

全称 mx.datacontainers.DataParams
命名空间 mx.datacontainers
类 DataParams
继承层次结构 DataParamsObject
源文件 ~\mx\scripts\datacontainers\DataParams.js
提供一个数据容器参数的包装类。 如果参数个数为1，则包装为数据查询参数，例如过滤条件等。 如果参数个数为2，则第一个参数包装为元数据查询参数，第二个参数包装为数据查询参数。

示例 1

以下为 DataParams 的示例。说明了如何使用 DataParams 类型封装查询参数
JavaScript 代码
// 如果数据表中已有四个字段，而只需要查询其中两个字段 "deptno,deptname" ，并且过滤出"deptno='0001'"的数据。 var dataParams = new mx.datacontainers.DataParams({columns: "deptno,deptname"},{filter: "deptno,deptname"}); var dataGrid = new mx.datacontrols.DataGrid({ //constructor }); dataGrid.load(dataParams);

构造函数

名称 说明
DataParams() 构建 DataParams 类的新实例。

字段

名称 说明
dataParams 获取或设置一个 JSON 对象，表示数据查询参数。例如：指定查询的过滤条件，{filter: "col1='soso'"}
metaParams 获取或设置一个 JSON 对象，表示元数据查询参数。例如：指定查询的数据列，{columns: "col1, col2"}
